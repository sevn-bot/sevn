# media_generation — MiniMax L2 specialist reference

Level-2 **`media_generator`** specialist backed by MiniMax REST APIs.
Operators pass **short intents** plus optional structured variables; the worker
augments with templates and returns **`trace`** metadata.

**Status:** MVP — correct payloads, unique artifacts, and adapter contracts under
mocks; optional live smoke via `SEVN_MEDIA_LIVE=1`. Not a production SLA guarantee.

## Quick start

```json
"subagents": {
  "specialists": {
    "media_generator": {
      "model": "minimax-3",
      "provider": "minimax",
      "assigned_to": ["tier_b"],
      "requestable_by": ["triager", "tier_b"],
      "max_concurrent": 2,
      "skill": "media_generation"
    }
  }
}
```

API key: `providers.minimax.api_key` or `MINIMAX_API_KEY` / `SEVN_SECRET_MINIMAX`.

### Wait policy

| Kind | Spawn `wait` |
|------|----------------|
| image / image_i2i / music / voice | `true` |
| video / video_i2v / video_s2v / video_fl2v / video_template | `false` (announce-back) |

Video generation can take minutes; cascade `wait=true` may kill the run mid-poll.

## API coverage matrix

| Kind | MiniMax API | Model | Required inputs |
|------|-------------|-------|-----------------|
| `image` | `/v1/image_generation` (T2I) | `image-01` | `prompt` |
| `image_i2i` | `/v1/image_generation` (I2I) | `image-01` | `prompt`, `reference_image` |
| `video` | `/v1/video_generation` (T2V) | `MiniMax-Hailuo-2.3` | `prompt` |
| `video_i2v` | `/v1/video_generation` (I2V) | `MiniMax-Hailuo-2.3` | `prompt`, `first_frame_image` |
| `video_s2v` | `/v1/video_generation` (S2V) | `S2V-01` | `prompt`, `subject_reference` |
| `video_fl2v` | `/v1/video_generation` (FL2V) | `MiniMax-Hailuo-02` | `prompt`, first + last frames |
| `video_template` | `/v1/video_template_generation` | Video Agent | `template_id` |
| `music` | `/v1/music_generation` | `music-2.6` | `prompt` |
| `voice` (clone) | upload + `/v1/voice_clone` | `speech-2.8-hd` | `source_audio` + literal `preview_text`/`speech_text` |
| `voice` (TTS) | `/v1/t2a_v2` | `speech-2.8-hd` | `voice_id` + literal `speech_text` |

`file_id` from video query is accepted as **int or str**. Downloads are capped at 100 MiB.
Local media paths must stay under the workspace `content_root`.

## Structured variables (lean)

Pass any subset. **Unset variables are omitted** from the augmented prompt (no filler defaults).
Unknown `template` slugs **raise** (no silent fallback to `default`).

| Variable | Example |
|----------|---------|
| `scene` | `"rainy Tokyo alley at night"` |
| `style` | `"watercolor illustration"` |
| `subject` | `"young woman with red scarf"` |
| `mood` | `"melancholic but hopeful"` |
| `lighting` | `"neon rim light"` |
| `camera` | `"[Push in] then [Pan left]"` |
| `genre` / `tempo` / `instrumentation` | music |
| `delivery` | voice character notes (trace only for TTS) |

---

## Voice: literal spoken text

`speech_text` and `preview_text` are sent **verbatim** to MiniMax.
Augmentation (`template`, `delivery`, `mood`) is recorded in `trace` only and is
**never** spoken. Clone requires one of those literal text fields.

Voice shorthand (`voice:…`) is rejected — use JSON.

---

## Artifacts

Filenames are `media-{kind}-{stem}-{uuid8}.{ext}` so identical prompts cannot collide.
The worker verifies the written file size matches generated bytes before returning
`artifact_path`.

## Trace output

```json
{
  "artifact_path": "channel_files/sess/media-image-fox-a1b2c3d4.jpg",
  "kind": "image",
  "trace": {
    "user_request": "fox in autumn",
    "template_key": "illustration",
    "augmented_prompt": "…",
    "variables": { "scene": "forest", "style": "watercolor", "user_request": "fox in autumn" },
    "api_model": "image-01"
  }
}
```

For voice TTS, `trace.spoken_text` is the literal utterance; `voice_character_notes`
holds the unused augmentation string.

## Not wired

Voice Design, lyrics-only API, music-cover, async long-form TTS, multi-image `n` > 1.

## Errors

| Message | Fix |
|---------|-----|
| `configure subagents.specialists.media_generator` | Add specialist block |
| `MiniMax API key missing` | Set key |
| `unknown template …` | Use `list_prompt_templates.py` |
| `path escapes workspace` | Keep files under content root |
| `voice clone requires preview_text or speech_text` | Provide literal spoken text |
| `voice shorthand unsupported` | Use JSON task |
