Settings reference

Common workspace options you may tune after setup.

This page is updated automatically when the product changes.

Most settings live in sevn.json in your workspace. The setup wizard sets sensible defaults; use Mission Control or edit the file directly for advanced changes. Secrets (API keys, tokens) should use secret references — never paste raw keys into this file.

SettingDescription
Message queue behavior
gateway.queue_mode		 When you send a new message while the bot is still replying, cancel stops the in-flight reply; steer waits to inject your new message at a safe point.
Gateway port
gateway.port		 TCP port for the web UI, API, and WebSocket chat. Default is 3001 on localhost.
Telegram channel
channels.telegram.enabled		 Turn the Telegram bot adapter on or off. Requires a bot token stored as a secret.
Reply quick actions
channels.telegram.quick_actions		 Show Regen, thumbs up/down, Share, and Feedback buttons under assistant messages in Telegram.
Text-to-speech mode
voice.tts_mode		 Control when the bot reads replies aloud — off, every reply, or only when you ask.
Triager model
models.triager		 The model that classifies your intent and routes the conversation to the right executor tier.
Long conversation memory
memory.lcm.enabled		 Keep lossless message history and summaries so the assistant can search past context.
Self-improvement jobs
self_improve.enabled		 Allow scheduled or manual jobs that propose small improvements to your workspace configuration.
