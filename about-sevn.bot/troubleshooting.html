Troubleshooting

Common problems and what to try first.

Bot does not respond in Telegram

  • Confirm the gateway is running: sevn doctor should report green checks.
  • Verify your bot token is stored correctly and Telegram channel is enabled in config.
  • Check gateway logs in your workspace logs/ directory.

Model errors or empty replies

  • Confirm your API key or subscription auth is set in secrets and the egress proxy is running.
  • Run sevn config validate to catch schema issues.
  • Try a different model in Mission Control or sevn.json.

Web UI will not load

  • Ensure you are using the URL and port from onboarding (default http://127.0.0.1:3001).
  • If behind a tunnel, confirm gateway.proxy_headers matches your reverse proxy setup.

Settings button does nothing in Telegram

Some menu buttons are visible but marked coming soon — see the settings preview for current availability. Buttons marked Available should respond immediately.

Still stuck?

Run sevn doctor --json and share the output (redact secrets) when asking for help in the community channels.
