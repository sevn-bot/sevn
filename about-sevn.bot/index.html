Welcome to sevn.bot

Your personal AI assistant — one bot on your machine, in the channels you already use.

sevn.bot is a personal assistant you run yourself. It remembers context across conversations, works with the model provider you choose, and can use tools and skills to get things done on your behalf.

What you can do

  • Chat from Telegram, the web UI, or the CLI
  • Adjust behavior from the Telegram /config menu — see the settings preview
  • Manage models, memory, and integrations from Mission Control when the dashboard is enabled
  • Extend capabilities with bundled skills and built-in tools — including planned Google Workspace (Gmail, Calendar, Drive, Sheets, Docs)
  • Run parallel work via sub-agents when gateway.queue_mode is multi
