Getting started

Install sevn.bot, complete setup, and send your first message.

Install

git clone https://github.com/sevn-bot/sevn.git
cd sevn
make setup

sevn onboard

make setup installs uv (when missing), fetches Python 3.12+ via uv, and installs dependencies. sevn onboard runs the setup wizard and registers gateway + proxy daemons by default.

Prefer the terminal UI? Run sevn onboard --cli instead of the default browser wizard.

Keep your install up to date

From inside your clone, pull the latest source and reinstall when you want a manual update:

sevn sync --latest

Remote VPS deploy

To provision a bot on a server you control, export a bundle with sevn export-secrets, copy deploy/inventory.example.toml to a local inventory file, then run make deploy-remote-check and make deploy-remote. See the operator runbook in the repository checkout.

First conversation

After the gateway is running, open Telegram and message your bot, or visit the web UI URL shown at the end of onboarding. Type naturally — the assistant routes your message, picks tools when needed, and replies in the same channel.

Useful commands in Telegram

  • /new — start a fresh conversation
  • /config — open the inline settings menu (preview here)
  • /help — short command list
  • /status — gateway health snapshot
